What is the difference between local storage and session storage?

LocalStorage and sessionStorage are comparable. Data in sessionStorage is deleted when the page session ends, however data in localStorage never expires. A unique page session is established and allocated to a certain tab in the browser each time a document is loaded there.

What is the difference between global scope and block scope in javascript?

Function Scope: A variable declared within a function is only used inside of that function and cannot be utilized outside of it.
Block Scope: When a variable is declared inside an if, switch, for, or while statement, it is only available within that statement.

how event loop work in javascript?

The sole task of the event loop is to keep track of the call stack and callback queue. The Event Loop will pull the first event from the queue and push it to the Call Stack, which will then execute it if the Call Stack is empty. In the Event Loop, such an iteration is referred to as a tick.
